# Telegram Video Downloader

A professional Flask-based web application that allows downloading public Telegram videos
using a message link with selectable quality.

## Features
- Paste Telegram message link
- Download public Telegram videos
- Select available video quality
- No ads
- Clean and simple UI

## Tech Stack
- Python
- Flask
- Telethon
- HTML, CSS

## Requirements
- Python 3.9+
- Telegram API ID & API HASH
